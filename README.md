# 代码命名工具

#### 介绍
java代码命名工具，基于开源库中的类名、变量名、方法名，为你推荐合适的命名

[https://jadepeng.gitee.io/code-naming-tool/](https://jadepeng.gitee.io/code-naming-tool/)。

#### 使用方法

打开[https://jadepeng.gitee.io/code-naming-tool/](https://jadepeng.gitee.io/code-naming-tool/)后，在中文输入框中输入 中文命名，然后回车即可。也可以直接在英文输入框输入英文，搜索候选。
